Git Bash Commands

  1. General
    • cd xxdir 前往某个目录下
    • cd .. 返回上级目录
    • ls 列出目录里的文件
    • cat xxfile 显示xxfile里的内容
    • pwd 显示当前目录的路径
    • history显示所有用过的command
    • mv xxfile newnamefile 改文件名
    • rm -rf xxdiv 删除某个目录
    • touch xxfile 新增xx文件
    • mkdir xxname 创建新文件夹xxname
    • git mv xxfile xxfolder/ 把xxfile移动到xxfolder里
    • ../ ../../ link里的文件地址上层文件夹或上上层文件夹
  2. Repository
    • git clone xxURL 从GitHub上copy需要clone的url下载到本地
    • git config --get remote.origin.url 查看GitHub的url
    • git add xxfile 把xxfile更新到缓存区
    • git commit -m “some messages” 确认所有更新
    • git add --all && git commit -m "comment" 更新缓存一次性完成，包括所有文件
    • git pull 把GitHub上的更新下载到本地
    • git push 把本地的更新上传到GitHub上 git push -f 强制更新remote
  3. Branch
    • git checkout xxbranch 转到xxbranch上
    • git push -set-upstream origin xx branch 把本地的branch推送到GitHub上
    • git fetch + git merge = git pull
    • git merge xxbranch 把xxbranch和main的内容合并
    • git log 显示commit的数量和内容 q退出
  4. Flask
    • flask --app hello run 在python里运行flask app hello.py
    • flask --app hello run --debug
  5. Django
    • 新建程序步骤： django-admin startproject xxproject *自动建一个项目文件夹 找到setting.py, 把'xxproject',添加到installed_apps里
